# setup-node-ts

yarn init -S

## Typescript e TSC

yarn add typescript -D
yarn tsc --init

## Express

yarn add express
yarn add @types/express -D

## ts-node-dev

yarn add ts-node-dev -D

## ESLint

yarn add eslint -D
yarn create @eslint/config
yarn add -D @typescript-eslint/eslint-plugin@latest @typescript-eslint/parser@latest
